# Complete-T-SQL-tutorial-for-MSSQL-Server
This repository contains my Complete (Beginner-to-Advanced) SQL source codes for SQL (T-SQL) training course for use with MSSQL Server, designed for Data Analysts, Data Engineers, Data Scientists, Software Engineers and MLEs.

## Author

`Adeyemi Adedoyin Simeon` | [LinkedIn](https://www.linkedin.com/in/adedoyin-adeyemi-a7827b160/)

## Features

- Complete and practical-oriented approach
- Multiple use-cases with examples
- Beginner-friendly, yet all-encompassing for even advanced users.
- From Beginner to Advanced level
- Focuses on the use of T-SQL (a Microsoft variant of SQL) for interacting with MSSQL Server DB

## Setups

- Download and install Microsoft SQL Server 2025 Express edition or latest from [Microsoft Official page](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)
- Download and install latest version of SQL Server Management Studio 22 (SSMS) from [Microsoft Official page](https://learn.microsoft.com/en-us/ssms/install/install)
